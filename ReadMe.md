# SYNTH

## Requirements
* Linux
* RtAudio library
* X11 library
* ALSA sound library


## Usage
Program that let's you play a synthesizer using your keyboard.

![Keyboard layout](/Keyboard.png)

Left/Right Arrow Keys: Change the Waveform
